#+DESCRIPTION: KernelSU / KernelSU-Next LKM for OnePlus sm8650 Android 16+ ROMs.
#+LANGUAGE: en

* About

This repository was created to build kernelsu.ko and kernelsu_next.ko to be utilized with the Android 16 ROMs for OnePlus sm8650 devices as [[https://github.com/realahnet/kernel_oneplus_sm8650][YAAP's Kernel]] is no longer GKI-compliant.

As such, upstream's kernelsu/kernelsu_next.ko will not work to root via the LKM method.
